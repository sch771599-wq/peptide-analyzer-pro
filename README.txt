# Анализатор Phi/Psi (MOL2)

Программный модуль на языке Python для расчета и визуализации торсионных углов Phi (φ) и Psi (ψ) на основе файлов формата .mol2 (включая ансамбли конформеров).

## Особенности
- Парсинг MOL2 файлов с сохранением оригинальной нумерации атомов.
- Интерактивный выбор атомов для расчета диэдрических углов.
- Визуализация структуры молекулы с помощью RDKit.
- Генерация карты Рамачандрана (Plotly).
- Экспорт результатов расчетов в формат Excel.

## Локальный запуск
1. Установите необходимые зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Запустите приложение:
   ```bash
   streamlit run app.py
   ```

## Использование в научной работе
Данное ПО разработано для анализа конформационной подвижности пептидов.
